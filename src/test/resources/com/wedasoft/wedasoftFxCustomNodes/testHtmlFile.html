Graph handling

Create a new graph

A new graph can be created via the menu item "Start/New".

Resize a graph

The size of the current graph can be altered via the menu item "Functions/Resize graph".

Save a graph

The current graph can be saved via the menu item "Start/Save".

Load a graph

A previously saved graph can be loaded via the menu item "Start/Load".

Delete a graph

Each graph can be deleted via the menu item "Start/Delete".

Node handling

Add a new node

A new node can be inserted into the graph via the menu item "Functions/Add node".

Open the nodes context menu

The context menu of a node can be opened by clicking it with your secondary mouse button.

Edit a node

A previously added node can be edited by its context menu via "Edit".

Delete a node

A previously added node can be deleted by its context menu via "Delete".
Deleting a node will lead into deleting all of its connections too.

Move a node

Each node can be moved on the graph by dragging it with the pressed primary mouse button.

Resize a node

Each node can be resized by dragging its resize icon with the pressed primary mouse button.

Place node on or behind another nodes

Each node can be placed on or behind another nodes by its context menu via "Put to front" and "Put to back".

Connection handling

Select a node

Each node can be selected by clicking "CTRL + Primary Mouse Button" on it.

Unselect a node

Each node can be unselected by clicking "CTRL + Primary Mouse Button" on it.

Unselect all nodes

All nodes can be unselected at once by clicking "ESC" on your keyboard.

Add a connection between two nodes

To connect two nodes you must select the two nodes first. Now open the context menu of a selected node and click on the menu item "Connect selected nodes".

Open the connections context menu

The context menu of a connection can be opened by clicking it with your secondary mouse button.

Delete a connection

Each connection can be deleted by its context menu via "Delete connection".

Other

Exit the application

The application can be exited via the menu item "Start/Exit".

Open user manuals

The user manuals can be opened via the menu item "?/User manuals".

Get application information

Information about the application can be displayed via the menu item "?/About".
